Records on the Wall

Records on the Wall incorporates an intuitive API powered live music calendar specifically built for the Charlotte area and a complete listing of supported venues along with links to venue websites, tickets, and maps. The API fires off the next 50 live shows in the Charlotte market.


jQuery, AJAX, and Bootstrap are the technologies used to parse the JSON data into the shows table. The site was built on a WordPress platform for future content flexibility.

  • Code: GitHub

My Little Alphabet Assistant

My little alphabet assistant was built to help children learn the alphabet along with the proper pronounciation. Each keyboard letter key is associated with a WAV file that triggers audio and visual components.


JavaScript is used throughout to fire off event handlers when the keys are pressed and CSS transitions are used to add visual effects to the application.

This application allows the user to manipulate the three initial variables set in CSS: border spacing around the image, the blur effect, and the border/font base color.

Flexible Panel Display

This flexible panel display shows the power of JavaScript along with CSS flex properties and CSS transitions. When a panel is clicked, the panel will flex out and transition in additional words initially hidden from view.


JavaScript event handlers are used for the click functionality which in turn triggers the addition of a CSS class that adds the flex and transition effects.

Custom HTML5 video player

This is a customized version of the HTML5 video player with hidden player controls along with skip and rewind functionality.


JavaScript is used to toggle between play/pause, rewind/skip functionality, volume control, and full screen action. CSS is used to hide controls and transition them in when moused over. I also used a custom linear gradient color scheme for the background.

Fun with HTML5 Canvas

This is a drawing application using the HTML5 canvas element that allows the user to draw with the mouse/mousepad.


This app is powered by JavaScript and ulitlizes the 2D context feature. The line type, width, and color are also set using JavaScript. One large draw function is used to create the line stroke and uses the associated X, Y coordinates from the canvas to determine the line path. Event listeners fire upon click of the mouse and also listen for the X, Y coordinates.
